# 🎬 AI Movie Insight Builder

An AI-powered movie analysis tool that fetches IMDb movie details and generates audience sentiment insights using AI.

---

## 🚀 Live Demo

👉 https://ai-movie-insight-six.vercel.app/

---

## 📌 Features

- 🔎 Search movies using IMDb ID (e.g., tt0133093)
- 🎥 Fetch movie title, poster, cast, rating & plot
- 🤖 AI-generated audience sentiment summary
- 📊 Sentiment classification (Positive / Mixed / Negative)
- 📱 Fully responsive modern UI
- ⚡ Deployed on Vercel

## 🧠 How It Works

1. User enters an IMDb movie ID.
2. OMDb API fetches movie metadata.
3. The backend processes movie data.
4. OpenAI generates sentiment summary.
5. App classifies overall audience sentiment.
